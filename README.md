# Review Site Frontend

---

A Vue-powered frontend website that consumes review data from our Strapi Headless CMS backend.
This site displays categorized reviews for Movies, Books, Games, and Music, featuring dynamic pages, search functionality, and responsive design.

---

## ‚ú® Features

* **Dynamic Content:** All reviews are fetched from Strapi REST API in real time
* **Category Filtering:** Browse reviews by Movies, Books, Games, or Music
* **Search Functionality:** Search reviews instantly by keywords
* **Single Review Pages:** Click any review to view its full content
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop

## üöÄ Quick Start

### Prerequisites

* Node.js 18+
* npm or yarn

### Local Development

1. Clone the repository:

```bash
git clone https://github.com/busa0019/Review-Site-FrontEnd
cd Review-Site-FrontEnd
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open your browser:

```bash
http://localhost:5173 (Vite default)
```

Enjoy!
